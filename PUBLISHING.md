# Public repository scope

This directory is published as the standalone `nguyenthevietquang07/band-forge`
repository. It is deliberately separate from the surrounding career workspace.

## Included

- BandForge application code, contracts, tests, and implementation docs.
- Title-free authored demo fixtures, including the `Am | F | C | G` progression
  used by the cycle demos.
- Deterministic export and verification artifacts generated from those fixtures.
- Required third-party sample/runtime attribution and provenance records.

## Excluded

- User-supplied event-chart scripts and every `beauty-and-the-beat-*` report or
  artifact.
- SQLite databases, local virtual environments, caches, and other machine
  state.
- Any future uploaded chart, lyric, score, MIDI, audio, or packet unless its
  publication rights have been independently confirmed.

## Publication checklist

1. Run the verification commands in the README.
2. Confirm `git status --ignored` shows private event material as ignored.
3. Review staged paths; do not use a blanket `git add -A` until the list is
   understood.
4. Keep claims limited to deterministic structural validation and the saved
   local demo evidence. Do not describe the MVP as a professional arranger,
   streaming transcriber, or trained model.
